<b><u>Description</u></b>

<i>ActivateByNum</i> enables Windows 7 task-switching hotkeys on XP and Vista.

<b>Windows Logo Key :windows:+1</b> switches to the first window on the taskbar and <b>Windows Logo Key :windows:+2</b> the second, etc. (It works with only the first 10 windows.) For a collapsed group of window buttons, repeatedly pressing the same <b>Windows Logo Key :windows:+n</b> combination (where <b>n</b> is the position of the group on the taskbar) will cycle through each window in the group.

The source code (AutoHotkey script) is available <a href="http://code.google.com/p/activatebynum/downloads/list">here</a>.

<b><u>Supported systems</u></b>

Windows XP (32-bit), Windows Vista (32-bit and 64-bit) 

<b><u>Other recommended utilities</u></b>

<i>ActivateByNum</i> works best when used with <a href="http://taskix.robustit.com/"><i>Taskix</i></a> or <a href="http://nerdcave.webs.com/"><i>Taskbar Shuffle</i></a> for rearranging taskbar buttons and <a href="http://sw2.deviantart.com/art/Iconize-Taskbar-Buttons-108127143"><i>Taskbaric</i></a> for iconizing taskbar buttons.
